Racing Timing Checklist

  1. Chronotrack Controllers
    1. Turn on controllers and manually set time using time.gov.
    2. Connect each controller via ethernet cable and broadcast time.
    3. Verify Controllers are visible within Fusion when filtering for today's date. If controller does not show up then reboot controller.
    4. Lay mats with antennas.
      1. Use short cords for mats 4 wide or less.
      2. Connect Pro2 with mini controller via ethernet cable.
      3. Setup Flash Points if required. The order of the cable connectors does not matter.
      4. Check antennas are working using signal card in controller case.
      5. Verfiy tag reads on the controller screen by using 999 tag.
  2. Fusion
    1. Open Fusion and drag controllers onto streams. Mini controllers are not visible from Fusion. The Pro2 controller connected to the mini via ethernet cable is responsible for uploading tag reads for both boxes (minis do not support cellular).
    2. Expand the sessions drop down menu and click on the icon to verify reads are showing up in Fusion.
    3. Verify Local Log and RaceDay Scoring are checked under the output tabs.
  3. RaceDay Scoring Application
    1. Start Reads connector.
    2. Check data issues.
    3. Reports -> Actions -> Filters -> uncheck "Show Non-Starters" for each race.
    4. Reports -> Actions -> Edit -> remove unnecessary columns (usually split and gun time).
  4. Typical Race Scoring Tasks
    1. Set gun time on controller.
    2. Begin playing results in Fusion
    3. Verfiy recent reads are showing up in Fusion
    4. Write down at least overall top 3 men and top 3 woman for each race distance.
    5. Resume report Auto-Saves in RaceDay Scoring.
    6. Verfiy results are showing up on Run Signup. Under Reports in RaceDay Scoring is the link for Live Results or you can find the race on Run Signup.
    7. Print Results.
    8. Check Raw Reads -> Ignored Reads for potential issues.
    9. Copy files from controllers to flash drive. RaceOps -> File Management -> Select Active.
    10. Export race from RaceDay Scoring and email it along with controller files to Ryan.
